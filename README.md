## Repositories

| Repo | Description |
|------|-------------|
| [openRigOS](openRigOS/) | Custom Linux image for ham radio hardware (RPi Zero 2W, RPi4, RPi5, x86_64) |
| [openRigConsole](openRigConsole/) | Desktop app (macOS, Linux, Windows) — rig control, logging, DX cluster |
| [openRigMobile](openRigMobile/) | Mobile app (iOS, Android) |
| [openrig_core](openrig_core/) | Shared Dart package — rig clients, ADIF, cluster, grid utils |
| [openRigWeb](openRigWeb/) | Project website |
